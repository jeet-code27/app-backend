Instructions & Tips

Getting Started:

  • • Update the Base URL if your server runs on different port
  • • Test connection first to ensure your server is running
  • • Use "Fill Sample Data" to quickly populate forms
  • • Check the API Response section for detailed results

Keyboard Shortcuts:

  • • Ctrl+1 - Create Request tab
  • • Ctrl+2 - Track Request tab
  • • Ctrl+3 - Admin Panel tab
  • • Ctrl+4 - Analytics tab

Testing Notes:

  • • MongoDB ObjectIds must be valid 24-character hex strings
  • • Phone numbers should be 10-digit Indian mobile numbers
  • • Email validation follows standard email format
  • • Request tracking uses the auto-generated requestId (REQ-...)

API Endpoints Covered:

  • • ✅ POST / (Create Request)
  • • ✅ GET /track/:requestId (Track Request)
  • • ✅ GET /admin (Get All Requests)
  • • ✅ PUT /admin/:id/status (Update Status)
  • • ✅ PUT /admin/:id/priority (Update Priority)
  • • ✅ All other admin endpoints included
